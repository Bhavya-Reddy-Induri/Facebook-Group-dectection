Instructions to run the code
---------------------------------------------
1. The Dataset folder consists of the dataset with all the features and elements of the Facebook Dataset and the Ego Networks.
2. Grouping and Preprocessing consists of the python files for converting the data in a singular sequence.
3. The node_information execl sheet consists of the processed data.
4. "Python 3.7.3" must be installed on the system to successfully run these files.
5. The python notebook files are the algorithms that have been implemented. They are -
    a. labelPropogation.ipynb
    b. fastUnfolding.ipynb
    c. spectralClustering.ipynb
6. All the .ipynb files can be run in Jupyter Notebook.
7. All the .py files can be run using "python filename.py".
